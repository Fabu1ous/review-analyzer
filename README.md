# Review Analyzer

API-сервис для автоматического анализа клиентских отзывов и помощи службе поддержки. Проект разрабатывается в рамках учебного курса.

## Суть проекта
Система принимает текст отзыва, обрабатывает его через LLM и возвращает строгий JSON, который содержит:
- `sentiment`: Тональность (позитив/негатив/нейтрально)
- `urgency`: Уровень срочности (от 1 до 10)
- `issues`: Список выявленных проблем
- `suggested_reply`: Сгенерированный ответ от лица компании

## Технологический стек
* **Язык:** Python 3.10+
* **Фреймворк:** FastAPI
* **Валидация данных:** Pydantic
* **LLM Интеграция:** OpenAI API / OpenRouter (модели с поддержкой JSON Mode)
